rulecatcher · the gate

Your AI never speaks
malformed again.

Show it a handful of examples and it learns the grammar. From then on, anything off-format is caught: a wrong-order token gets rotated back into place; a foreign token is rejected outright.

→ The gate that keeps the model inside its own syntax — so the structure you designed is the structure you get.

rulecatcher catch examples.txt --scope mydsl
rulecatcher lint  newtext.txt  --scope mydsl
# non-zero exit on a violation → CI-gateable

honeyc · the compiler

Write it once.
Read it as anything.

Express an idea in one compact glyph chain, then render it as prose, triples, Prolog, or a database query — whichever the moment needs.

→ One source of truth, every shape. Stop maintaining the same idea in four formats by hand.

honeyc render examples/mbr.rune --as prose
honeyc render examples/mbr.rune --as cypher
# 🌸‍💧 → 🍯 → 🍹  becomes whatever you need
